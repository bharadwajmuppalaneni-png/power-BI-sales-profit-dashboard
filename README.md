# Power BI Sales & Profit Dashboard

## Objective
To analyze sales performance, profitability, and regional trends using interactive visualizations.

## Tools Used
- Power BI
- Excel

## Key Features
- Sales trend analysis
- Regional performance comparison
- Profit and margin calculation
- Interactive dashboard

## Key Insights
- South region contributes highest revenue
- Product A dominates overall sales
- West region shows lower profit margins
- Opportunity to reduce costs and improve margins

## Business Impact
Helps businesses:
- Identify high-performing regions
- Improve profitability
- Make data-driven decisions
